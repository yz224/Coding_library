关于我们：

渡鸦科技（北京）有限责任公司诞生于 2014 年 5 月，是微软加速器第五期校友， Y Combinator W15 唯一的中国大陆创业团队。

用艺术诠释科技，一直是渡鸦科技的愿景。这支年轻的团队致力于打造下一代人机交互和人人交互系统，并为其命名为 Project Flow 。而渡鸦科技工程师偶然以 Project Flow 中的音乐模块做出来的小玩具乐流（MusicFlow），作为整个项目的缩影，在不经意间引领了一代音乐播放器的设计和交互风格。

「It all flows together」未来手机或者智能终端的交互的方式就是运用“流”，而渡鸦科技希望做到的，是打破落后的信息壁垒，去掉一切繁琐的外衣，去掉 App 的形式外壳，使所有交互呈现去壳化的状态，直接解决人类最本质的需求。

目前，渡鸦科技在北京侨福芳草地、北京南锣鼓巷以及硅谷均设有办公室。
